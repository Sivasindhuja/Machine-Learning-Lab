
## Machine Learning Lab – Implementations

This repository contains implementations of basic Machine Learning concepts and algorithms as part of my coursework. The goal of this work was to understand how different algorithms behave on real datasets and to practice the complete ML workflow.

---

## What is covered

### Data preprocessing

* Handling missing values (Titanic dataset)
* Removing duplicates
* Basic statistical analysis (mean, median, min, max)
* Encoding categorical data using:

  * One-hot encoding
  * Label encoding

### Dataset handling

* Loading datasets using `sklearn`
* Fetching datasets from UCI repository
* Saving datasets as CSV files
* Splitting data into training and testing sets

### Algorithms implemented

* Logistic Regression
* Decision Tree
* Random Forest
* k-Nearest Neighbors (k-NN)
* Naive Bayes
* K-Means Clustering

---

## Datasets used

* Breast Cancer dataset
